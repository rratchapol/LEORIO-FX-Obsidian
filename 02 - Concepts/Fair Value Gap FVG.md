# Fair Value Gap FVG

FVG คือพื้นที่ imbalance ที่ราคาวิ่งผ่านเร็วจนเหลือช่องว่างเชิงโครงสร้าง ผู้สอนใช้เป็นโซนที่ราคามีโอกาสย้อนกลับมาทดสอบหรือเป็นเหตุผลกลับฝั่งเมื่อเห็นบริบทสนับสนุน

## ใช้งานอย่างไร

- ใช้เป็น checklist ก่อนเข้าออเดอร์ ไม่ใช่สัญญาณเดี่ยว
- เช็คบริบทจาก market structure ก่อน แล้วค่อยหา entry trigger
- กำหนดจุดผิดทางและขนาดความเสี่ยงก่อนกดออเดอร์
